# Medical Imaging Research Prerequisites

1. [Medical Imaging Systems](MI.html) : X-ray imaging, CT, MRI, UCA, and basic Ultrasound imaging
2. [Ultrasound Imaging](US.html) : From 3D wave equations to Hystotrypsy
3. [Biostatistics](Biostats.html) : Pre-requisite for any empirical research really; t-tests, ANOVA, normality tests, correlation, and more.
4. [Causal Inference](CA.html) : When just correlation isn't enough and you want to make a statement; Expected Outcomes, DiD, TWFE, DAG analysis, confounding and colliding factors. 
5. [Computer Vision](CV.html) : Essential for Image processing; SIFT, Edge and corner detection, convolutional filters, image stitching (Homography), etc.
